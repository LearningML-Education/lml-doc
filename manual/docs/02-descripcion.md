# Descripción de la plataforma LearningML

Tres partes component la plataforma *LearningML*:

- La página web destinada a la difusión y descripción de la propia herramienta y de cuestiones relacionadas con el Machine Learning y su aprendizaje. Se accede a través de la url: <a href="https://learningml.org" target="_blank">https://learningml.org</a>. 

- El *editor de modelos*; una aplicación para la construcción de modelos de Machine Learning. Se accede a través de la url <a href="https://v2.learningml.org" target="_blank">https://v2.learningml.org</a>. Presenta dos modos de uso: básico y avanzado. El primero recomendado para primaria y primer ciclo de secundaria, y el segundo, indicado a partir del segundo ciclo de secundaria, ya que permite adquirir una comprensión más profunda de los algoritmos de Machine Learning.

- El *editor de programación*; una plataforma de programación por bloques con la que se puede desarrollar cualquier tipo de aplicación informática. Se trata de una modificación de la aplicación open source [*Scratch*](https://scratch.mit.edu/) a la que se le han añadido nuevos bloques para usar los modelos de Machine Learning construidos con el *editor de modelos*. En esta versión, como novedad, con el *editor de programación* también se pueden construir modelos sin necesidad de usar el *editor de modelo*, es decir, de manera autónoma usando los bloques de LearningML. Al *editor de programación* se accede desde el *editor de modelos* o directamente a través de la url: <a href="https://v2.learningml.org/scratch" target="_blank">[https://v2.learningml.org/scratch](https://v2.learningml.org/scratch)</a>.
