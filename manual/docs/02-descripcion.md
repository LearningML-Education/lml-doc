# Descripción de la plataforma LearningML

Tres partes component la plataforma *LearningML*:

- La página web destinada a la difusión y descripción de la propia herramienta y de cuestiones relacionadas con el Machine Learning y su aprendizaje. Se accede a través de la url: <a href="https://learningml.org" target="_blank">https://learningml.org</a>. 

- El *editor de modelos*; una aplicación para la construcción de modelos de Machine Learning. Se accede a través de la url <a href="https://lmlplus.learningml.org" target="_blank">https://lmlplus.learningml.org</a>. Presenta dos modos de uso: básico y avanzado. El primero recomendado para primaria y primer ciclo de secundaria, y el segundo, indicado a partir del segundo ciclo de secundaria, ya que permite adquirir una comprensión más profunda de los algoritmos de Machine Learning.

- El *editor de programación*; una plataforma de programación por bloques con la que se puede desarrollar cualquier tipo de aplicación informática. Se trata de una modificación de la aplicación open source [*Scratch*](https://scratch.mit.edu/) a la que se le han añadido nuevos bloques para usar los modelos de Machine Learning construidos con el *editor de modelos* o para construir y usar modelos de manera autónoma, es decir, prescindiendo del *editor de modelo*. Se accede a ella desde este último o directamente a través de la url: <a href="https://lmlplus.learningml.org/editor" target="_blank">https://lmlplus.learningml.org/editor</a>.
